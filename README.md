Alur Pengambilan KRS :

Login -> sebagai Admin -> Tampilan Pilihan Data Mahasiswa atau Data Mata Kuliah -> Tampilan Menu Input Data ->  Masukkan data sesuai kategori -> Tombol Tambahkan -> Tampilan pada Tabel -> Selesai.
